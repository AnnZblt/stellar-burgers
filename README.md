# Космическая бургерная Stellar Burgers 
Система управления с маршрутизацией и авторизацией

## Описание проекта
Проект представляет собой веб-приложение с системой маршрутизации, управлением состоянием и механизмом авторизации пользователей. Приложение включает защищенные маршруты и личный кабинет.

## Основные особенности
   * Система маршрутизации с поддержкой защищенных маршрутов
   * Управление состоянием через глобальные данные
   * Механизм авторизации с защитой личного кабинета
   * Юнит-тестирование компонентов приложения
   * Интеграционное тестирование бизнес-логики

## Технические детали

### Этап 1: Базовая функциональность
   * Маршрутизация:
      * Реализация системы роутинга
      * Настройка защищенных маршрутов
      * Управление навигацией
   * Состояние приложения:
      * Внедрение глобального состояния
      * Подключение к страницам
      * Синхронизация данных
   * Авторизация:
      * Механизм входа/выхода
      * Защита личного кабинета
      * Проверка прав доступа

### Этап 2: Тестирование
   * Юнит-тесты(Jest):
      * Инициализация rootReducer
      * Тестирование слайса burgerConstructor
      * Тестирование слайсов (на примере ingredients)

   * E2E-тесты(Cypress):
      * Настройка моковых данных
      * Добавление ингредиентов в конструктор
      * Работа модальных окон
      * Создание заказа
